# 처음 시작하기 — Claude Code 설치부터 첫 대화까지

## Claude Code가 뭔가요?

Claude Code는 **AI 비서가 내 컴퓨터에서 직접 일하는 도구**입니다.

### 일반 ChatGPT와 뭐가 다른가요?

| 구분 | 일반 ChatGPT | Claude Code |
|------|----------|------------|
| **어디서?** | 웹 브라우저에서만 | 내 컴퓨터에서 직접 |
| **뭘 할 수 있나?** | 물어보고 답 받기 | 파일을 직접 읽고 수정, 명령어 실행, 검색 |
| **파일 수정** | 불가능 | 직접 가능 |
| **프로젝트 이해도** | 낮음 (파일을 못 봄) | 높음 (전체 파일 분석) |

**쉽게 말해:** "ChatGPT가 내 컴퓨터에 눈과 손을 가진 셈"이라고 생각하면 됩니다.

---

## 1단계: 설치하기

### Mac에서 설치

1. **터미널**(컴퓨터에 글자로 명령하는 검은 화면) 열기
   - 맥의 오른쪽 위 검색 아이콘 → "터미널" 입력 → 엔터

2. 아래 한 줄을 복사해서 터미널에 붙여넣고 엔터 치기

```bash
brew install anthropic/claude/claude
```

3. 혹시 위가 안 되면 이 방법 시도하기

```bash
npm install -g @anthropic-ai/claude
```

4. 설치 확인하기

```bash
claude --version
```

✅ `Claude v0.x` 같은 버전이 나오면 설치 완료!

### Windows에서 설치

1. **명령 프롬프트** 열기 (또는 PowerShell)
   - 시작 → "cmd" 입력 → 엔터

2. 아래 명령어 복사-붙여넣기

```bash
npm install -g @anthropic-ai/claude
```

❌ 만약 `npm을 찾을 수 없음` 에러가 나면?
→ Node.js를 먼저 설치해야 합니다. https://nodejs.org 에서 LTS 버전 설치

## 2단계: 처음 실행하기

### 로그인하기

1. 터미널에서 아래 입력:

```bash
claude
```

2. 이런 메시지가 보일 것입니다:

```
Claude Code v0.x
Type /login to authenticate
```

3. 이렇게 입력:

```
/login
```

4. 엔터 누르면 **자동으로 브라우저 열림**
5. Anthropic 계정으로 로그인 (없으면 가입)
6. 승인하기
7. 터미널로 돌아가면 완료!

### 프로젝트 폴더에서 시작하기

Claude Code를 쓸 때는 항상 **프로젝트 폴더에서 시작**해야 합니다.

```bash
cd ~/내_프로젝트_폴더_경로
claude
```

❓ "폴더"가 뭐냐고요?
- 아, 내 컴퓨터에 파일들이 들어있는 상자 같은 거라고 생각하면 됩니다.
- `cd`는 "이 폴더로 이동해" 라는 의미입니다.

## 3단계: 첫 대화 나누기

이제 AI한테 뭔가 물어보세요!

### 예시 10가지

**상황 1: 이 프로젝트가 뭐 하는 건지 모를 때**

```
이 프로젝트 구조 설명해줄래?
```

Claude Code가 폴더 구조를 보고 자동으로 설명해줍니다.

**상황 2: 파일 목록을 보고 싶을 때**

```
프로젝트에 파일이 뭐가 있어?
```

**상황 3: 특정 파일이 뭐 하는 건지 모를 때**

```
src/auth.js가 뭐 하는 파일이야?
```

**상황 4: 코드가 이해 안 될 때**

```
이 코드가 뭐 하는 건지 쉽게 설명해줄래? src/utils/helper.js
```

**상황 5: 뭔가 고장 났을 때**

```
로그인이 안 되는데 src/login.js 확인해줄래?
```

**상황 6: 새로운 기능을 만들고 싶을 때**

```
프로필 사진 업로드 기능 만들어줄래?
```

**상황 7: 코드가 너무 복잡할 때**

```
이 코드 좀 더 간단하게 정리해줄래? src/processing.js
```

**상황 8: 웹에서 뭔가 찾고 싶을 때**

```
React의 최신 버전이 뭐야? 어떤 변화가 있었어?
```

**상황 9: 에러 메시지가 나올 때**

```
이 에러 메시지가 뭔 의미야? "TypeError: undefined is not a function"
```

**상황 10: 파일을 찾고 싶을 때**

```
로그인 관련 파일이 어디에 있어?
```

## 4단계: 모드란?

Claude Code는 3가지 **모드**(자동차 기어처럼 생각하면 됨)가 있습니다.

| 모드 | 설명 | 언제 쓰나 |
|------|------|----------|
| **default (D)** | AI가 파일 수정할 때마다 "이거 할까요?" 묻기 | 처음 쓰는 사람, 중요한 작업 |
| **plan (P)** | AI가 읽기만 하고 수정 안 함 (분석만) | 코드 분석, 구조 파악 |
| **bypassPermissions (S)** | AI가 마음대로 파일 수정 (자동 승인) | 신뢰할 수 있는 작업, 빠르게 진행 |

**처음엔 default만 쓰면 됩니다!**

모드 바꾸는 법:

```bash
claude --mode plan
```

또는 대화 중에 이렇게 말하기:

```
모드를 plan으로 바꿔줘
```

## 5단계: 자주 쓰는 명령어

대화 중에 이 명령어들을 쓸 수 있습니다.

| 명령어 | 뜻 |
|--------|------|
| `/help` | 사용 가능한 명령어 목록 보기 |
| `/clear` | 지금까지의 대화 모두 지우고 새로 시작 |
| `/status` | 지금 어떤 상태인지 확인 |
| `/exit` | Claude Code 끝내기 |

예를 들어:

```
/clear
```

엔터 치면 — 지금까지 대화한 모든 내용이 사라지고 새로 시작합니다. (헷갈릴 때 쓰세요)

## 6단계: 끝내기

### 간단하게 끝내기

```bash
Ctrl + C
```

또는:

```
/exit
```

엔터 치면 끝!

## 💡 팁

✅ **꼭 기억하세요!**
- Claude Code를 실행할 때는 항상 프로젝트 폴더에서 `cd` 해서 이동한 후 `claude` 입력
- 파일 경로를 말할 때는 정확하게 (예: `src/components/Button.js`)
- 못 이해한 것처럼 물어보기 (AI가 더 쉽게 설명함)
- 자주 `/clear` 써서 대화 초기화하기
